# **App Name**: Athena Justice

## Core Features:

- Secure Authentication: Role-based access control with secure login for police officers, counselors, and administrators.
- Sheeter Profile Management: Create, read, update, and delete (CRUD) operations for managing rowdy sheeter profiles, including personal details, criminal history, and behavioral tags.
- AI Counseling Prep: Profile analysis to determine counseling focus and suggest questions using criminal history and behavior tags, operating as a decision support tool.
- Voice-to-Voice Interface: Allows counselor to speak and have their speech transcribed, stores recorded responses from the sheeter, and reads dynamically generated questions aloud.
- Automated Report: System auto-generates interaction reports containing emotional indicators, cooperation levels, trend behavior, risk reassessment, and session summaries.
- Interactive Dashboard: Provides a high-level overview of counseling progress, with key metrics to include high-risk individuals, repeat offenders, and trends.
- Encrypted Data Storage: Ensures all data is stored with full encryption, audit logging and tamper-proof security in Firestore database.

## Style Guidelines:

- Primary color: Dark blue (#30475E) to evoke trust and authority.
- Background color: Light gray (#F2F4F7), offering a neutral and clean backdrop.
- Accent color: Teal (#5F9EA0) for important interactions.
- Body and headline font: 'Inter', a grotesque-style sans-serif, provides a modern and objective look for readability.
- Use of government-style icons with a focus on clarity and simplicity.
- Clean, minimal layout with clear labels and instructions, designed for accessibility.
- No flashy animations; use only subtle transitions to maintain a calm and authoritative tone.
